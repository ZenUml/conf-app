Pre-Edit Gate: Modal Variants & Their Behaviors

When a Lite-app user attempts to edit an existing macro in a space with ≥100 macros (CSS enabled, unpaid), the PageEditorPaywallGate mounts instead of the editor. It shows one of four modal variants based on persona & tenant size. This page documents the new explicit "Continue editing" button, what each modal action does, and which actions grant edit access.

● Grants edit access ● Does NOT grant access ● Opens external URL ● Modal close (no-op at gate)

1. Which modal shows?

UpgradePromptRouter.vue selects a variant based on the PERSONA_AWARE_PAYWALL flag, the user's persona (creator vs bystander, threshold M=5), and tenant-size estimate.

PAP off → LegacyPrompt
PAP on · personalAuthored < 5 → BystanderNotice
PAP on · tenantSize unknown → ComparisonView
PAP on · creator + size known → HeavyCreatorPrompt

2. Modal variants (with new Continue editing button)

Each modal now has a labeled Continue editing without upgrading action — the only path that grants edit access. Every other button/action either opens an external URL, dismisses the modal (no-op at the gate), or notifies an admin.

4. Before / after the bypass fix

Codex's P1 #4 review caught that the prior code translated any close event into "continue-editing" — so Escape, backdrop, ×, "Got it", "Remind me later", and even the upgrade CTAs all silently mounted the editor.

Before
Every dismissal mounted the editor
// PageEditorPaywallGate.vue
<UpgradePromptRouter
  ...
  @close="$emit('continue-editing')"
/>
  • Escape → editor mounts
  • Backdrop click → editor mounts
  • × → editor mounts
  • Marketplace CTA click → editor mounts (worst case: pays nothing, gets editor)
After
Only the explicit button mounts the editor
// PageEditorPaywallGate.vue
<UpgradePromptRouter
  ...
  @continue-editing="$emit('continue-editing')"
/>
  • Escape / backdrop / × → modal stays in place
  • Upgrade CTAs → external URL opens, modal stays
  • "Continue editing without upgrading" button → editor mounts + paywall_continued_editing fires
  • Stage 4 can now measure abandonment as a real signal
